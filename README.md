# Shop Management App

This project helps connect multiple shops of the same company, creating a chain of shops connected through the same website.

## Users

### Shop Cashier
- Add customer details
- Generate bills

### Shop Owner
- Add or delete shops
- View customer details
- Generate reports

## Tech Stack
- **Backend:** Java, Spring Boot  
- **Frontend:** HTML, CSS  
- **Build Tool:** Maven  
- **Database:** MySQL
